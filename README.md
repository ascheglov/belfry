# Belfry the SSH client

Belongs to public domain.

Usage:

    belfry [-h] [-p port] [-bastion host] destination [command]

Flags:

    -bastion string
        bastion host (SSH to destination from this host)
    -p string
        remote port (default "22")
    -h
        help message
